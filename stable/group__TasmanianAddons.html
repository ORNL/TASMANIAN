Detailed Description

Extra Capabilities
The Addon module of Tasmanian offers a series of templates that offer additional capabilities not necessarily included in the core modules. The templates sit in a separate module for various reasons, e.g.,
  • some methods are hard to classify or address fringe use cases
  • the templates depend on third-party libraries and should not overwhelm the core modules with dependencies
