Detailed Description

Class and data types used for dynamic construction of Global grids.

Author
Miroslav Stoyanov

Defines the storage class and struct types used for construction of Global grids. The class is more complex than other examples because rules with growth of more than one point need to collect several samples before the grid can be extended with another tensor.
